# Google Stitch AI - High Fidelity Vibe Prompts

These prompts are designed to be copied and pasted directly into **Google Stitch** (`stitch.withgoogle.com`) to generate matching, responsive HTML/Tailwind CSS blocks.

---

## 🎨 Global Prompt Context (Pre-pend to all generations)
> **Aesthetic Context:** Design a premium, futuristic glassmorphic UI matching a space-cyberpunk vibe. Background is deep space black (#000000 / #0a0612), with glass containers using thin borders (`rgba(255,255,255,0.08)`) and high blur (`backdrop-filter: blur(20px)`). The text uses slate white (`#f8fafc`) for main and muted grey (`#94a3b8`) for descriptions. Accent colors are hot-pink (`#d946ef`), glowing purple (`#a855f7`), and electric cyan (`#06b6d4`). Fonts: Poppins (headers, heavy weights) and Outfit (body).

---

## 💻 1. Main Dashboard & Workspace (Tab Layout)
> Create a full-screen, grid-based dashboard for the "Instagram Invader AI" image generation tool. The layout should split into two columns on desktop, single column on mobile. 
> Left side: A configuration card. Include:
> - Drag-and-drop zone grid for "Subject Portrait" and "Instagram Profile UI Screen".
> - Custom horizontal selector buttons for tearing presets (Circle, Vertical, Horizontal, Diagonal) using glass pill icons.
> - A text area for the breach caption.
> - An interactive horizontal range slider for "Cinematic Glow" (0-100%).
> - A toggle pill switch for UI Theme (Dark vs Light).
> - A primary glowing pink/purple button "INVADE NOW" with a bolt icon.
> Right side: Live Preview sandbox displaying a before/after split image comparison slider centered inside a glowing frame. Include download and share buttons below.

---

## 🔐 2. Premium Authentication Screen (Login / Register Card)
> Create a centralized, full-screen glassmorphic login and registration dialog.
> Features:
> - Header: "INSTAGRAM INVADER AI" logo with a hot-pink text shadow glow, and subtitle.
> - Double tabs: "Sign In" and "Register" with custom active slide indicators.
> - Interactive input groups (Email, Password, Name) containing thin grey borders that transition to glowing cyan on focus, and custom vector icons inside.
> - Standard "Google Sign-in" button featuring the Google logo, styled with a semi-transparent glass border and hot-pink glow effect on hover.
> - Clear visual error alerts that slide down in dark red/crimson.

---

## 🎟️ 3. Pricing & Limits Upgrade Screen (Card Grids)
> Design a pricing model modal for upgrading from free tiers. Give it a highly premium, "Instagram Creator Pro" vibe.
> Structure:
> - Three glassmorphic cards: Free ($0), Pro ($9/mo), and Creator Bundle ($29/mo).
> - Pro card: Highlighted with a pulsing neon purple border (`box-shadow: 0 0 25px rgba(168,85,247,0.5)`) and a "Most Popular" floating badge.
> - Features list inside cards: Custom checkboxes showing features like: "4K High Resolution Composites", "Custom Tear Vector Import", "Batch processing API", and "Ad-free experience".
> - Hover micro-animations on all buttons.

---

## ✂️ 4. Image Cropper & Orientation Canvas Modal
> Design an overlay/modal component for fine-tuning user-uploaded photos.
> Features:
> - Large centered square workspace with dynamic grid lines (rule of thirds).
> - Drag handles at the corners of the crop frame.
> - Bottom alignment bar: Circular buttons for "Rotate Left", "Rotate Right", "Flip Horizontal", and "Aspect Ratio Presets" (1:1, 4:5, 16:9).
> - Action button row: "Cancel" (muted grey text button) and "Apply Crop" (electric cyan button with a checkmark).
